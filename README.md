# PROYECTO-MARKETING-CAMPA-AS-SQL-POWER-BI

## Sobre el proyecto

Este proyecto fue desarrollado como un ejercicio integral de limpieza de datos utilizando SQL, con un enfoque educativo que simula un escenario de trabajo en el mundo real. El objetivo principal es aplicar buenas prácticas de data cleaning para transformar un conjunto de datos en bruto en información consistente, estructurada y lista para su análisis.

A lo largo del proyecto se realizan diferentes etapas del proceso de limpieza de datos, incluyendo exploración inicial del dataset, detección y eliminación de duplicados, estandarización de formatos, tratamiento de valores faltantes y eliminación de columnas innecesarias.

Cada paso del proceso está documentado y acompañado por las consultas SQL utilizadas, con el fin de mostrar de forma clara y reproducible cómo se identifican y resuelven problemas comunes de calidad de datos.
